# MyShows.me для Kodi (`plugin.video.myshows`)

Синхронизация просмотренных сериалов и фильмов — и ваших оценок — из Kodi в
[MyShows.me](https://myshows.me).

> Неофициальный аддон. Не связан с MyShows.me и не одобрен сервисом.

---

## Возможности

- **Автоотметка** (фоновый сервис): отмечает серию/фильм просмотренным по
  достижении заданного порога; после выхода из плеера предлагает поставить оценку.
- **Просмотр и управление** прямо из Kodi:
  - **Новые серии** — вышедшие, но непросмотренные серии тех сериалов, что вы смотрите.
  - **Мои сериалы** — по статусам (смотрю / буду смотреть / бросил / просмотрено).
  - **Мои фильмы** — просмотрено / буду смотреть.
  - **Поиск** сериалов и фильмов; отметка, оценка, смена статуса; отметка всего сезона.
- Галочка «просмотрено» ставится только когда просмотрены **все** серии
  (и у сериала, и у сезона).
- Графическое окно оценки звёздами (1–5).

## Требования

- Kodi 19 (Matrix) или новее — Python 3 (`xbmc.python` 3.0.0).
- Аккаунт на MyShows.me.

## Установка

**Из zip:** скачайте `plugin.video.myshows-<версия>.zip` из раздела
[Releases](https://github.com/temandroid/plugin.video.myshows/releases),
затем в Kodi: *Дополнения → Установить из zip-файла*.

> Внимание: встроенный распаковщик Kodi строже системного и иногда не
> справляется с потоковыми архивами исходников от GitHub/Gitea
> («Failed to unpack archive»). Ставьте именно zip из **Releases** (там верная
> структура), а не «Download ZIP» с кнопки Code.

**Вручную:** скопируйте файлы аддона в папку с именем ровно
`plugin.video.myshows` внутри каталога `addons/` вашего Kodi и перезапустите Kodi.

## Настройка

Откройте настройки аддона, введите логин и пароль MyShows и нажмите *Войти*.
Опции автоотметки: вкл./выкл., порог просмотра, запрос оценки, статус «Смотрю»
при старте, резолв по названию (fallback).

## Как это работает с MyShows

JSON-RPC поверх OAuth2 (password grant). Сериалы — эндпоинт `v2`, фильмы — `v3`.
Сопоставление идёт сначала по внешним ID (IMDB/TVDB для сериалов,
IMDB/Kinopoisk для фильмов), затем — по точному совпадению названия. OAuth-клиент —
публичный демо-клиент MyShows (`apidoc`), секретов в коде нет. Токен хранится
только локально в профиле аддона.
